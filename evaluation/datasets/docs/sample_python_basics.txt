Python Basics for Live RAG Evaluation

Recursion is a technique where a function calls itself to solve a problem by breaking it into smaller versions of the same problem. A recursive function should always define a base case.

The base case is the stopping condition. It prevents the function from calling itself forever and keeps the recursion from running until the interpreter reaches a recursion limit.

Python lists are mutable. A list can be changed after it is created by appending new items, removing items, replacing existing items, or reordering the values.

Because lists are mutable, two variables can refer to the same list and both see the updates. This is different from immutable values like numbers and strings.
